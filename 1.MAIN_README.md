#  MEDIA_ANALYSIS PROJECT

This project focuses on analyzing **Facebook Ads** and **LinkedIn Posts** using datasets in `.xlsx` format. We explored each dataset using **three different BI tools** ‚Äî **Microsoft Excel**, **Power BI**, and **Tableau** ‚Äî to build six interactive dashboards and extract valuable marketing and engagement insights.
